# Cutie Cat Tarot — card face images

วางภาพหน้าไพ่ของ **Cutie Cat Tarot** ในโฟลเดอร์นี้ ตั้งชื่อไฟล์เป็นเลข 2 หลัก
(`01.png` … `78.png`) ให้ตรงกับหมายเลขไพ่ในตารางด้านล่าง แล้วภาพจะขึ้นในแอป
อัตโนมัติ (แท็บ "ความหมายไพ่", หน้าเปิดไพ่, ตัวอย่างในหน้าเกี่ยวกับ Deck)

- ไฟล์ที่รองรับ: `.png` (แนะนำสัดส่วนแนวตั้ง ~19:28 เหมือนไพ่จริง)
- ใบไหนที่ยังไม่มีไฟล์ จะแสดง **หลังไพ่** (`/img/card-back-cutie-cat.png`) แทน
  โดยอัตโนมัติ ไม่มีภาพแตก — ค่อย ๆ ทยอยเติมได้
- ใช้ชื่อไฟล์เดียวกันทั้ง thumbnail และภาพใหญ่ (ระบบใช้ไฟล์เดียวต่อไพ่)
